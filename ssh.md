# SSH Setup

## Goal

Control Mac mini terminal from iPhone using SSH.

## Mac settings

System Settings → General → Sharing → Remote Login ON

## Commands

Check username:

```bash
whoami
```
## Check IP address:

```bash
ipconfig getifaddr en1
```
## SSH format

```bash 
ssh username@ip-address
```
